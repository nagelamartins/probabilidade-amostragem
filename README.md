# Exercícios de Probabilidade e Amostragem 📊

Este repositório contém a resolução de uma lista de exercícios práticos desenvolvida durante o **Bootcamp Data Analytics da WoMakersCode**. 

## 🎯 Objetivo do Projeto
O objetivo principal é aplicar conceitos de probabilidade e amostragem utilizando uma base de dados fictícia da força de trabalho da população brasileira.

## 🛠️ Tecnologias e Ferramentas Utilizadas
- **Python**
- **Pandas & NumPy**: Manipulação e tratamento dos dados.
- **Matplotlib & Seaborn**: Visualização de dados e análise de distribuições (KDE/Histogramas).
- **SciPy (stats)**: Modelagem estatística e cálculo de distribuições (Binomial e Normal).

## 🧠 Conceitos Estatísticos Aplicados
- Cálculo de Probabilidade Complementar e Condicional.
- Função Densidade de Probabilidade (PDF) e Distribuição Acumulada (CDF).
- Distribuição Normal (Média, Variância e Desvio Padrão).
- Distribuição Binomial aplicada a grandes populações.
- Cálculo de Margem de Erro Amostral para proporções populacionais.

## ✅ Conclusão
A resolução destes exercícios permitiu colocar em prática conceitos essenciais de probabilidade e amostragem utilizando Python. Além de realizar os cálculos estatísticos, foi possível interpretar os resultados e compreender como essas técnicas podem ser aplicadas na análise de dados. Esses fundamentos são importantes para desenvolver análises mais confiáveis e embasar decisões orientadas por dados. Além disso, servem como fundamentos para estudos mais aprofundados sobre o tema no futuro.
